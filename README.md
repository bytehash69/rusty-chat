A very simple Rust client/server chat program.

## Quick start
- Install Rust from https://rustup.rs
- Clone this repo:
  - `git clone https://github.com/bytehash69/rusty-chat`
  - `cd rusty-chat`

## Run
- Run the server:
  - `cd server`
  - `cargo run`
- Run the client:
  - `cd client`
  - `cargo run`

## How it works (brief)
- This repository contains a basic server and client written in Rust.
- Start the server, then run one or more clients to connect and chat.
